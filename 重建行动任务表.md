## 技术栈确认

### 后端技术栈
- **运行时**: Node.js 18+
- **语言**: TypeScript
- **框架**: Express.js
- **数据库**: PostgreSQL + Prisma ORM
- **缓存**: Redis
- **AI服务**: 阿里云百炼API + DeepSeek API
- **认证**: JWT

### 前端技术栈
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **状态管理**: Zustand
- **UI组件**: Ant Design
- **HTTP客户端**: Axios
- **图表库**: Recharts

### API集成插件
- **框架**: React + TypeScript
- **UI组件**: Ant Design
- **代码编辑器**: Monaco Editor
- **HTTP客户端**: Axios
